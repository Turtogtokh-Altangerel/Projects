Calorie Trainer is a user-friendly application on command prompt, which is recommended to
anyone who would like to track their daily food consumption to reach their goals. The program
can add any number of new users, records of whom are saved; therefore, existing users can
simply start the program by entering their names. The program can also track daily nutritional
intakes for a user to analyze on tracking of past dates or get a reminder of the remaining
number of recommended calories and protein for the day. Moreover, users can look up
nutritional values of food in the food list and in case of not finding, the user can add the
information into the list easily. After a user makes a progress, such as losing weight, the user
can change a measurement, which updates and recommends a new number of recommended
daily intake.
